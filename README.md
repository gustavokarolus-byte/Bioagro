# Bioagro
O Bio Agro é um projeto inovador que une produtividade agrícola e sustentabilidade ambiental por meio da economia circular. Sua proposta consiste em coletar resíduos orgânicos gerados na cadeia de alimentos, transformá-los em adubo orgânico de alta qualidade e devolvê-los ao campo como insumo para a produção agrícola.
